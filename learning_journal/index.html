Forms

The article mentioned a lot of different good practices people should follow when creating a form design for their website. It talked a lot about visual cues that help the user when they are filling out the form.

The first thing it mentions is to have the input fields on one column and to not have too many input fields, but if unavoidable, use multi-step forms.

It also mentions that it is good practice to label what the input is supposed to be, and provide some examples to what it should be outside of the input field.

The gist of the article was to make it so we provide the user with visuals that help them figure out what to do/whether things were successful, etc.
